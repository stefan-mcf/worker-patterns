# Changelog

All notable changes to this project are documented here.

## Unreleased

The repository is public and prepared for the 0.1.0 release line. A formal GitHub release/tag has not been published yet.

## 0.1.0 - Prepared initial public release

- Added dry-run worker-pattern selection for runtime execution shapes.
- Added CLI entrypoint: `worker-pattern`.
- Added stdio MCP entrypoint: `worker-pattern-mcp`.
- Added policy-driven lane/profile hints.
- Added execution-plan and prompt rendering.
- Added optional temp-home smoke checks for MCP/runtime integration.
- Added public documentation, contributor guide, security policy, and MIT license.
- Documented the temporary `hermes_worker_patterns` compatibility window and policy synchronization gate.
